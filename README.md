# Deliverr

Liquor, beer, wine, spirits, and tobacco delivery across Regina, Saskatchewan.
19+ only. ID required on delivery.

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Capacitor (iOS & Android)
- Supabase (database, auth, storage, edge functions)

## Local development

```bash
npm install
npm run dev
```

## Mobile builds

After editing native assets in `resources/`, sync platform projects:

```bash
npm run mobile:sync
```
